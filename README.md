🏦 Banking Customer Analysis Dashboard

End-to-End Data Analytics Project using Python, SQL, Power BI, Excel, and Tableau

📌 Project Overview

The Banking Customer Analysis Dashboard is an end-to-end data analytics project designed to analyze customer demographics, banking products, income distribution, and financial behavior. The project demonstrates the complete analytics workflow, from data extraction and cleaning to SQL analysis, exploratory data analysis (EDA), and interactive dashboard creation.

The objective is to transform raw banking data into meaningful business insights that support customer segmentation, product strategy, and business decision-making.

🎯 Business Objectives
Analyze customer demographics and banking behavior.
Understand customer income distribution.
Identify relationships between occupation, nationality, gender, and banking products.
Explore customer account balances and financial products.
Build an interactive dashboard for business stakeholders.
Generate actionable insights to improve banking services.
